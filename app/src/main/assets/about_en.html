About Brain As Assistant Project

 

This project has two main goals. The first is Supplying user a convenient mobile tool for keeping record about what he plans to do. And second goal is Timely notification about tasks.

 

This program give user an opportunity to combine advantage of mobile app and at the same time use comfortable web-interface on www.brainas.net.You can create a task sitting at your notebook or desktop pc and this one imidetly immediately appears in your device with installed mobile app of BA.

 

BA doesn`t spend user's time on boring registration procedure You can login in web-app or mobile app via your Google account.

 

You can use app just as a todo-list or add time and GPS condition for task.For example, if you want to buy a lemons you can add the task "Buy a lemon" with GPS coordinates of fruit shop. When you will be there you will get a notification from BA about lemons task.

 

Using of Google maps makes adding coordinates an easy thing

 

User has fast access to tasks that were activated by time or GPS-coordinates. All active task are available on first screen immediately after start of app.

 

Every task can be associated with picture. In this case user will see image instead of title on tile. Image is more sensitive way to represent task when user not have enough time for reading titles on small smartphone's screen

 

BA App give a tool for fast add image to task using Google image search.

 

This is only beta version and many things will be done in future like categories of tasks and abilities adding tasks for other people, for example your colleagues, friends and families.
